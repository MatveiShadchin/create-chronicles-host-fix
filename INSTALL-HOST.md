# Быстрая установка для хоста (без Cursor)

## 1. Бэкап
Скопируй `saves\ИмяМира` → `saves\ИмяМира_backup`

## 2. Скачай
https://github.com/MatveiShadchin/create-chronicles-host-fix/archive/refs/heads/main.zip

## 3. Установи
Распакуй и скопируй в `%appdata%\.minecraft\`:
- `kubejs` → заменить папку целиком
- `defaultconfigs` → заменить папку целиком

## 4. Перезапуск
Полностью закрой и открой Minecraft.

## 5. Проверка
JEI → поиск `Rotation Mechanism`
